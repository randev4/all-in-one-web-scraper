# URL Unshortener and Cleaner

This Python script helps to unshorten URLs (e.g., `https://reddit.com/`, `https://youtube.com/`) by following redirects and cleaning the final URL. It can also filter out unnecessary parameters, leaving only the essential ones.

## Features

- Unshortens any shortened URL to its final destination.
- Removes unnecessary URL parameters for a cleaner, more concise link.
- Allows customization of which parameters to keep.
